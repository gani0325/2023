# 프론트에서 서버에 데이터 요청하기

## 1) react

: HTML을 이쁘게 만들어주는 툴

- npx create-react-app .
- App.js 만들고 개발하기 npm run start
- 개발 끝난다면 -> npm run build
- -> html 파일 (main page) 이 생성
- fetch, axios 사용하여 데이터 요청하기
  - npm install axios

## 2) server

- server.js 만들기
- npm init -y
- npm install express nodemon cors
- server 실행하기 -> nodemon server.js
